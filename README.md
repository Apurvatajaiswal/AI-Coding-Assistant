#  AI Coding Assistant

An AI-powered coding assistant that captures your screen, extracts code using OCR, and detects errors with voice feedback.

---

##  Demo

![App Screenshot](screenshot.png)

---

##  Features

*  Screen Capture
*  OCR Text Extraction
*  Error Detection
*  Voice Feedback
*  Modern UI with animations

---

##  Tech Stack

* React
* FastAPI
* Tesseract OCR
* Framer Motion

---

##  Run Locally

### Backend

cd backend
uvicorn app:app --reload

### Frontend

cd frontend
npm install
npm start
